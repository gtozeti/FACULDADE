Exemplos de como montar links com Thymeleaf

Para todos os exemplos, comparar o HTML do template Thymelaf com o HTML referente à página apresentada no navegador após o processamento

Em geral, os links internos da aplicação são acessados através da expressão no formato @{ ... }. O Thymeleaf irá adicionar o valor do context-path da aplicação antes do nome do recurso.

Para carregar arquivos /css/estilos.css e /js/funcoes.js do diretório /resources/static - usar th:href="@{/css/estilos.css}" e th:src="@{/js/funcoes.js}" .
